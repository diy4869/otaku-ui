CoDesign Iconfont Demo

Symbol (推荐用法)：
  • 支持多色图标，支持调整颜色大小；
  • 兼容性支持 IE9+。
Class：
  • 不支持多色图标，支持按字体方式调整颜色大小；
  • 需要对图标进行标准化处理，兼容性支持 IE8+。
Unicode：
  • 不支持多色图标，支持按字体方式调整颜色大小；
  • 需要对图标进行标准化处理，兼容性支持 IE6+。

如引用Unicode或Class字体包，需要对上传的图标进行标准化处理，避免显示异常。

  • 闭合：图标路径描点要处于封闭状态
  • 合并：如果有多个图形进行组合，要对图形合并扩展
  • 轮廓化：要将描边转化为闭合图形，并填充颜色

详情可见帮助中心：https://codesign.qq.com/hc/icons/upload/

Unicode 引用

1. 添加 @font-face 块, 引用图标字体文件；

@font-face {
  font-family: 'iconfont';
  src: url('iconfont.eot');
  src: url('iconfont.eot?#iefix') format('embedded-opentype'),
  url('iconfont.woff2') format('woff2'),
  url('iconfont.woff') format('woff'),
  url('iconfont.ttf') format('truetype'),
  url('iconfont.svg#iconfont') format('svg');
}

2. 设置 .iconfont 的基本样式；

.iconfont {
  font-family: "iconfont" !important;
  font-size: 16px;
  font-style: normal;
  -webkit-font-smoothing: antialiased;
  -moz-osx-font-smoothing: grayscale;
}

3. 应用图标，输入对应的 Unicode 值。

<span class="iconfont">&#xe624;</span>

"iconfont" 是图标库下的 “iconfont family” 字段，可以通过编辑图标库查看，默认是 "iconfont"。

Symbol 引用

1. 引入 iconfont.js 文件，向页面添加 SVG sprite；

<script src="./iconfont.js"></script>

2. 设置基本样式；

<style>
.icon {
  width: 1em;
  height: 1em;
  vertical-align: -0.15em;
  fill: currentColor;
  overflow: hidden;
}
</style>

3. 应用图标，使用对应的 id 值。

<svg class="icon" aria-hidden="true">
<use xlink:href="#prefix-xxx"></use>
</svg>

"iconfont" 是图标库下的 “iconfont family” 字段，可以通过编辑图标库查看，默认是 "iconfont"。
"prefix-" 是图标库下的“iconfont 前缀”字段，可以通过编辑图标库查看，默认是 "icon-"。
